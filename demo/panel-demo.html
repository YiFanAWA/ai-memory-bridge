AI Memory Bridge v0.3.0

连接 Obsidian 笔记到 Claude Code 的 MCP 桥接插件。拖拽策展，TF-IDF 语义搜索，双向读写。

📥 安装

# 1. 拷贝插件到 vault
cp -r ai-memory-bridge/ ~/YourVault/.obsidian/plugins/

# 2. 在 Obsidian 设置 → 社区插件 → 启用 AI Memory Bridge

# 3. 配置 Claude Code MCP
# 在面板中点击 📋 配置 按钮复制 JSON，或手动添加：
{
  "mcpServers": {
    "ai-memory-bridge": {
      "command": "node",
      "args": ["mcp-bridge.js", "--vault", "/path/to/your/vault"]
    }
  }
}
